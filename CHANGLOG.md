# Changelog

## v1.2.0 - January 25, 2026

### Added
- **Options Page**: New settings page allowing users to customize their experience
  - Option to disable the context menu item on YouTube
  - Option to hide the "Watch on Captionz" button on YouTube video pages
- **Browser Action**: Quick access toolbar button to open the current YouTube video directly on Captionz
- **Footer Links**: Options page now includes links to Programming N' Language community and Captionz website

### Fixed
- Improved styling and positioning of the "Watch on Captionz" button on YouTube video pages

## v1.0.1

### Added
- Captionz-ext signature displayed on the Captionz website when videos are opened through the extension
- Automatic caption detection and enablement on YouTube videos

### Fixed
- Caption state detection to ensure subtitles are enabled when needed

## v1.0.0

### Initial Release
- Context menu integration for opening YouTube videos on Captionz
- Inject "Watch on Captionz" button on YouTube video pages
- Background service worker for managing extension functionality
- Support for Chrome, Edge, and Firefox browsers
